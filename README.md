# flappio

A flappy bird clone that uses html canvas.

![](/flap.gif)

## Contribution

- Make sure you have [node](https://nodejs.org/en/download)
1. **Run the frontend**

```
node server.js
```

2. **Run the backend** 
(in another terminal)
```
node Backend/backend.cjs
```

The game runs at http://localhost:3000
